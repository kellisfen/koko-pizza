# Dodo Pizza Analog — MVP план

## Концепция
Приложение для заказа пиццы с доставкой: клиент выбирает пиццу → оформляет заказ → отслеживает в реальном времени → курьер доставляет.

---

## Стек (MVP)
- **Frontend (клиент):** Next.js 14 (веб), React Native (мобильные)
- **Админ + Курьер:** Next.js (веб-панель)
- **Backend:** FastAPI (Python)
- **База данных:** PostgreSQL
- **Infrastructure:** VPS (Selectel/Timeweb)
- **Хостинг:** Vercel (frontend) + свой сервер (backend)
- **Карты:** Яндекс.Карты API
- **Платежи:** ЮKassa (СБП)

---

## MVP фичи

### Клиентское приложение
- [ ] Просмотр меню (пиццы, напитки, допы)
- [ ] Добавление в корзину
- [ ] Оформление заказа (адрес, время доставки)
- [ ] Оплата онлайн (СБП)
- [ ] Трекинг заказа в реальном времени

- [ ] Профиль пользователя

### Админ-панель
- [ ] Управление меню (добавить/удалить/редактировать)
- [ ] Просмотр всех заказов
- [ ] Изменение статуса заказа
- [ ] Статистика (заказы, выручка)

### Курьерское приложение
- [ ] Список доступных заказов
- [ ] Приём заказа
- [ ] Навигация по адресу (Яндекс.Карты)
- [ ] Отметка о доставке

### Бэкенд
- [ ] API для клиента, админа, курьера
- [ ] База данных PostgreSQL
- [ ] Push-уведомления (Firebase)
- [ ] Геолокация + расчёт расстояния/стоимости
